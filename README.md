# E- commerece Dashboard Power BI
Dashboard Power-BI


![dashboard](https://github.com/RutujaAjagekar/E-commerece-Dashboard-Power-BI/blob/main/Screenshot%202025-11-08%20222105.png)

Power BI project demonstrating end-to-end data analytics: from data cleaning to interactive dashboards.

## Overview

E-commerce data into actionable business intelligence with data modeling, DAX calculations, and interactive visualizations.

## Technical Skills Demonstrated

**Data Preparation**
- Power Query: importing, cleaning, and transforming CSV data
- Data validation and handling missing values
- String manipulation and standardization

**Data Modeling**
- Star schema design with fact and dimension tables
- Relational database architecture
- DAX measures: KPIs, YoY growth calculations, conditional formatting

**Dashboard Design**
- Interactive multi-visual dashboards with cross-filtering
- Custom map charts for geographical analysis
- Professional color schemes and visual hierarchy

**Tools Used**
- Power BI Desktop, Power Query, DAX
- Natural Earth Data, MapShaper, PowerPoint (wireframing)

## Dashboard Visualizations

- **KPI Cards:** Color-coded performance indicators (green/red) for Sales, Quantity, Customers, Average Price
- **Custom Map Chart:** Geographical sales distribution by country with interactive filtering
- **Rounded Bar Charts:** Revenue and quantity trends with error bars for YoY comparison
- **Line Charts:** Monthly sales and average price trend analysis
- **Area Charts:** Customer growth trends over time
- **Donut Chart:** Top 5 customers by sales
- **Bar Charts:** Top 5 units sold and top 5 countries by revenue
- **Year-over-Year Slicers:** Dynamic time-based filtering across all visuals

## Key Features

- 15+ DAX measures for KPIs and growth metrics
- Conditional formatting with color-coded indicators
- Custom geographical mapping
- Advanced chart types with professional formatting
- Dynamic cross-filtering between visualizations

## Project Workflow

1. **Data Preparation:** Import and clean multiple data sources
2. **Data Modeling:** Build relationships between fact and dimension tables
3. **DAX Calculations:** Write measures for KPIs and growth metrics
4. **Dashboard Design:** Wireframe layout and create interactive visuals
5. **Testing:** Validate interactivity and data accuracy
